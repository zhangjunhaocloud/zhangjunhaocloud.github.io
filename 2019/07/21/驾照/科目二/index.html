    上车后注意事项

    1. 调整左右两边后视镜，需要看到后车轮
    2. 空调调整到1档，这样不容易死火
    3. 带好安全带
    4. 调整座椅
    5. 按指纹

    第一项：倒车入库

    1. 沿着左起第二条黄线右边沿缓慢行驶(方向盘中间正对黄线右边)，直到左后视镜过了黄线部分停车
    2. 挂倒档，看左后视镜 后车轮刚压到地上的裂缝时，往右打死方向盘，继续倒车，直到看到第一个角的时候，方向盘回半圈，继续倒车，直到车身平行，方向盘回正，继续倒车，注意看后视镜中后车轮位置，刚刚压到地上裂缝的时候停车
    3. 挂1档，缓慢前行，当左前窗户刚刚过黄线的时候往左打1圈90度，继续前进，直到左后视镜过黄线停车
    4. 挂倒档，缓慢倒车，直到刚刚过延长线时，往左打死方向盘，待车身平行时，回正方向盘，注意看后视镜中后车轮，刚压到裂缝的时候停车
    5. 挂1档，缓慢前进，直到车身压过最前面的黄线的时候右转弯，进入下一个考试项目

    第二项：侧方位停车

    1. 沿着裂缝前进，直到右车窗后边沿刚刚到杆的位置时停车
    2. 挂倒档，方向盘向右打1圈，缓慢倒车，注意看左边后视镜
    3. 刚刚看到第二个角的时候，回正方向盘，注意看后视镜中，车轮位置，刚压到线(或快要压到线的时候)，向左打死方向盘
    4. 注意看右边后视镜，注意不要出线，车身进来之后停车
    5. 挂1档，打转向灯，缓慢前行，前车中间压到最前面黄线时回正方向盘，3分之2压黄线时往右打1圈，缓慢前进进入下一个考试项目

    第三项：曲线行驶

    1. 没什么特别要注意的，注意前车沿压线行驶即可，最后的时候要调整方向，沿着杆行驶

    第四项：直角转弯

    1. 注意看左边的杆，当肩膀与杆平行时，向左打死方向盘即可，进行下一考试环节

    第五项：坡道定点停车

    1. 先沿着白色箭头往前行，注意与右边距离，在刚上坡时，刹车多松一点
    2. 看不见白色箭头时，注意沿着前边黄色第二，第三中间行驶
    3. 注意看地下裂缝，当肩膀与地线重合时，停车，拉手刹，松手刹，方向盘稍稍左打
    4. 待车身抖动三下之后，缓慢松开刹车，前行通过考试
